Identity

1. whoami

Displays the username of the current effective user.

whoami

2. id

Prints real and effective user and group IDs.

id username

3. w

Shows who is logged on and what they are doing.

w

▼ Session Tracking

4. last

Shows a listing of last logged in users.

last -n 10

5. who

Displays a list of users currently logged into the system.

who -u

User and Group Management

1. sudo

Execute a command as another user, typically the superuser (root).

sudo apt update

2. useradd / userdel

Create a new user or delete an existing user account.

sudo useradd -m newuser
sudo userdel -r olduser

3. passwd

Change a user's password.

sudo passwd username

4. groupadd

Create a new group.

sudo groupadd developers

5. usermod

Modify a user account (e.g., adding a user to a group).

sudo usermod -aG sudo username

Permissions and Ownership

1. chmod

Changes the file mode bits (permissions).

chmod 755 script.sh
chmod +x script.sh

2. chown

Changes the owner and/or group of a file.

sudo chown user:group file.txt

3. su

Switch user to become another user (or root).

su - username

▼ Advanced

4. umask

Sets the default file creation permissions for the current shell.

umask 022

5. visudo

Safely edits the sudoers file to manage administrative privileges.

sudo visudo
