Connectivity Checking

1. ping

Sends ICMP ECHO_REQUEST to network hosts to check connectivity.

ping google.com

2. traceroute

Tracks the path that packets take to a network host.

traceroute google.com

3. mtr

Combines ping and traceroute into a single real-time diagnostic tool.

mtr google.com

▼ Advanced

4. telnet

User interface to the TELNET protocol; often used to check if a specific port is open.

telnet 192.168.1.1 80

5. nc (netcat)

The "Swiss-army knife" of networking. Used for reading/writing data across network connections.

nc -zv 127.0.0.1 22

6. nmap

Network exploration tool and security / port scanner.

nmap -sV 192.168.1.1

Network Configuration

1. ip addr

Displays and manipulates routing, network devices, interfaces and tunnels (Replaces ifconfig).

ip addr show

2. ip route

Show or manipulate the IP routing table.

ip route show

3. netstat

Prints network connections, routing tables, and interface statistics.

netstat -tuln

4. ss

Utility to investigate sockets. Faster and more detailed than netstat.

ss -p

▼ Advanced

5. tcpdump

Command-line packet analyzer. Allows you to intercept and display TCP/IP packets.

sudo tcpdump -i eth0

6. nmcli

Command-line tool for controlling NetworkManager and reporting network status.

nmcli device status

7. ethtool

Display or change ethernet card settings (Speed, Duplex, Wake-on-LAN).

sudo ethtool eth0

8. route

Older utility to show/manipulate the IP routing table.

route -n

DNS and Lookup

1. host

DNS lookup utility for finding IP addresses of domains and vice-versa.

host github.com

2. dig

DNS lookup utility for interrogating DNS name servers (More detailed than host).

dig github.com ANY

3. nslookup

Querying Internet name servers interactively.

nslookup google.com

4. whois

Client for the whois directory service (Domain registration info).

whois google.com

Data Transfer

1. curl

Transfer data from or to a server (Supports HTTP, FTP, etc.).

curl -I https://google.com

2. wget

Non-interactive network downloader.

wget https://example.com/file.zip

3. scp

Secure copy (remote file copy program using SSH).

scp file.txt user@remote:/path/to/

4. rsync

Fast, versatile, remote and local file-copying tool (Supports delta sync).

rsync -avz local_dir/ user@remote:backup/

▼ Advanced

5. sftp

Interactive file transfer program using SSH.

sftp user@remote_host

6. ftp

Internet file transfer program (Standard but insecure).

ftp ftp.example.com
