File Bundling

1. tar (Create)

Bundles multiple files/directories into a single archive file (tape archive).

tar -cvf archive.tar file1 file2 dir1

2. tar (Extract)

Extracts files from a tar archive.

tar -xvf archive.tar

3. tar (List)

Lists the contents of an archive without extracting it.

tar -tvf archive.tar

▼ Advanced

4. tar (Append)

Adds new files to an existing archive.

tar -rvf archive.tar newfile.txt

5. tar (Exclude)

Archives a directory while skipping specific files or subdirectories.

tar --exclude='*.log' -cvf backup.tar /var/www

6. cpio

An older utility used to copy files to and from archives (similar to tar but works with pipes).

find . | cpio -ov > archive.cpio

Linux File Compression

1. gzip / gunzip

The standard Linux compression tool. Fast but lower compression ratio.

gzip file.txt
gunzip file.txt.gz

2. tar + gzip (Combined)

Bundles and compresses in one command. Uses the '.tar.gz' or '.tgz' extension.

tar -czvf backup.tar.gz /home/user/docs

3. bzip2 / bunzip2

Slower than gzip but provides better compression. Uses '.bz2'.

bzip2 large_file.iso
bunzip2 large_file.iso.bz2

4. xz / unxz

Provides the highest compression ratio available in standard Linux. Very slow to compress.

xz huge_file.db
unxz huge_file.db.xz

▼ Specialized Compression

5. zcat / zless

Allows you to view the contents of a compressed text file without unzipping it.

zless logfile.gz

6. tar + xz

The modern standard for distributing source code.

tar -cJvf project.tar.xz /src

7. lz4

Extremely fast compression/decompression, used where speed is more important than size.

lz4 file file.lz4

ZIP and 7-Zip Formats

1. zip

Creates a ZIP archive, compatible with Windows and macOS.

zip -r bundle.zip folder_name

2. unzip

Extracts files from a ZIP archive.

unzip bundle.zip -d /target/path

▼ Cross-Platform Tools

3. 7z

High compression ratio format (7-Zip). Handles almost all archive types.

7z a archive.7z files/

4. rar / unrar

Handles the proprietary .rar format often used in large file sharing.

unrar x movie.rar
