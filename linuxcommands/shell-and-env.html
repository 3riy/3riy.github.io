Variables and Export

1. export

Sets an environment variable that will be passed to child processes.

export MY_VAR="Hello World"

2. echo $VARIABLE

Displays the value of a specific variable.

echo $PATH
echo $USER

3. env / printenv

Lists all currently defined environment variables.

printenv | grep SHELL

▼ Advanced

4. unset

Removes a variable from the current shell session.

unset MY_VAR

5. set

Lists all variables (including local ones) and shell functions.

set | less

Shell Environment

1. alias

Creates a shortcut for a longer command.

alias ll='ls -la'

2. source / .

Executes a file in the current shell context (often used to refresh .bashrc).

source ~/.bashrc

3. which

Shows the full path of the executable that runs when a command is typed.

which python3

▼ Configuration Files

4. ~/.bashrc

The per-user interactive shell configuration file.

nano ~/.bashrc

5. /etc/environment

System-wide environment variable definitions.

cat /etc/environment

Shell Shortcuts and History

1. history

Displays the list of previously executed commands.

history | tail -n 20

2. !! (Bang Bang)

Runs the very last command again (often used with sudo).

sudo !!

3. CTRL + R

Recursive reverse search through your command history.

(Press Ctrl+R, then start typing a command)

4. CTRL + A / CTRL + E

Move cursor to the beginning (A) or end (E) of the line.


5. CTRL + L

Clears the terminal screen (same as the 'clear' command).


6. !$

Refers to the last argument of the previous command.

mkdir my_folder
cd !$
