Disk Usage and Filesystems

1. df

Displays the amount of disk space available on filesystems.

df -h

2. du

Estimates file space usage in a directory.

du -sh /var/log

3. mount

Mounts a filesystem so it can be accessed by the system.

sudo mount /dev/sdb1 /mnt/usb

4. umount

Unmounts a currently mounted filesystem.

sudo umount /mnt/usb

5. lsblk

Lists information about all available or the specified block devices (disks).

lsblk -f

▼ Advanced

6. blkid

Locates/prints block device attributes like UUIDs and Labels.

sudo blkid

7. fsck

Checks and repairs a Linux filesystem. (Use with caution on unmounted disks).

sudo fsck /dev/sda1

8. sync

Flushes file system buffers to disk to prevent data loss.

sync

9. mkfs

Used to build a Linux filesystem on a device, usually a partition.

sudo mkfs.ext4 /dev/sdb1

Disk Partition Management

1. fdisk

A dialog-driven program for creation and manipulation of partition tables.

sudo fdisk -l

2. parted

A partition manipulation program supporting many partition table formats (GPT/MBR).

sudo parted /dev/sda print

3. gdisk

An interactive text-mode program for manipulating GPT partitions.

sudo gdisk /dev/sda

▼ Advanced

4. pvdisplay / vgdisplay / lvdisplay

Shows information about Physical Volumes, Volume Groups, and Logical Volumes (LVM).

sudo lvdisplay

5. resize2fs

Resizes ext2/ext3/ext4 file systems after changing partition size.

sudo resize2fs /dev/vg0/lv_data

6. swapoff / swapon

Disables or enables devices and files for paging and swapping.

sudo swapon -s

Hardware Information

1. lspci

Lists all PCI buses and devices in the system (e.g., GPU, Network cards).

lspci -v

2. lsusb

Lists USB buses and the devices connected to them.

lsusb -t

3. lshw

A detailed hardware lister that provides a summary of all hardware components.

sudo lshw -short

4. dmidecode

Retrieves hardware info (Serial numbers, BIOS versions) from the DMI table.

sudo dmidecode -t system

5. free

Displays the amount of free and used memory in the system.

free -h

▼ Advanced Hardware Inspection

6. smartctl

Control and monitor self-monitoring, analysis, and reporting technology (SMART) on HDDs/SSDs.

sudo smartctl -a /dev/sda

7. sensors

Displays current readings of all sensor chips (Temperatures, Voltages, Fans).

sensors

8. hdparm

Get/set SATA/IDE device parameters; often used to test drive speed.

sudo hdparm -Tt /dev/sda

9. lsdev

Displays information about installed hardware including IRQ, DMA, and I/O ports.

lsdev

10. nvme

Utility for managing and querying NVMe SSDs (requires nvme-cli).

sudo nvme list
